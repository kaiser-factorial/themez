Animated Level Backgrounds

Six animated background patterns inspired by the CatchFall game level themes (SKETCH.js). Each pattern uses pure CSS animations — no JavaScript required.

Level 1 — Mod Stripes

Animated color stripes cycling through red, blue, and yellow with vertical grid lines. The stripes scroll continuously.

.bg-mod-stripes
.bg-mod-stripes { /* 3-color scrolling stripes + grid lines */ }

Level 2 — Bezier Clock

Concentric circles with pulsing animation. Inspired by the bezier chord connections in the game.

.bg-bezier-clock
.bg-bezier-clock { /* Pulsing concentric circles */ }

Level 3 — Cantor Field

Recursive horizontal lines with symmetric top/bottom halves. Gentle vertical drift animation.

.bg-cantor-field
.bg-cantor-field { /* Recursive lines + center divider */ }

Level 4 — Halftone Rings

Concentric dot patterns in a grid. Three offset dot layers create a halftone effect with subtle movement.

Level 5 — Wave Marks

Sinusoidal rows of dots and dashes. Diagonal drift animation mimics wave motion.

.bg-wave-marks { /* Diagonal wave drift pattern */ }

Level 6 — Primary Grid

Grid with colored diagonal rectangles. Opacity pulse animation. Inspired by the game's geometric grid with colored cells.
