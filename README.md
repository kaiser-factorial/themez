## Quick Start

Each theme is a single CSS file. Just include it and use the class names:

```html
<link rel="stylesheet" href="primary_theme/components/primary-theme.css">
<body class="theme-primary">
  <button class="btn">Primary Action</button>
  <button class="btn btn-red">Danger</button>
  <span class="badge badge-solid-blue">Active</span>
</body>
```

## Design Languages

**Primary Theme** follows Bauhaus principles — primary colors only, geometric purity, functional minimalism, hard shadows, no rounded corners.

**Cyber Theme** follows terminal/hacker aesthetic — deep black backgrounds, neon green glow, monospace throughout, CRT scanline effects.
